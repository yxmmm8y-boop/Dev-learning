# 🐙 Git

Раздел с материалами по Git — системе контроля версий, используемой для управления изменениями в проектах и работы с GitHub.

---

## 📚 Содержание

* 📄 `baza.md` — основные понятия Git
* 📄 `commands.md` — основные команды
* 📄 `branches.md` — работа с ветками
* 📄 `merge.md` — слияние и конфликты
* 📄 `workprocess.md` — типичный процесс работы

---

## 🎯 Цель раздела

* Понять принципы работы Git
* Освоить базовые и продвинутые команды
* Научиться работать с GitHub в реальных задачах

---

## ⚙️ Что изучается

* контроль версий
* staging area и commits
* ветки и merge
* работа с удалёнными репозиториями
* разрешение конфликтов

---

## 🔄 Базовый workflow

```bash
git pull
git status
git add .
git commit -m "описание изменений"
git push
```

---

## 📌 Примечание

Материалы в папке являются частью общего DevOps-обучения и дополняются по мере изучения новых тем.
