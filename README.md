### DevOps Learning Repository
 Этот репозиторий - моя практическая база знаний по DevOps.
 По сути это подробный конспект о всех моих шагах в изучении специальности и сборник практических работ

## Структура

- Ansible - Автоматизация и управление конфигурациями 
- Docker - Контейнеризация приложений
- Git - контроль версий и работа с GitHub
- Kubernetes - оркестрация контейнеров
- Networking - сетевые основы и протоколы
- Projects - практические проекты
- Terraform - инфраструктура как код


## Цель
 
* освоить  DevOps-инструменты с 0
* научиться работать с инфраструктурой 
* сделать портфолио из проектов
