#Git_теория
 - Система контроля версий 
Задача - упрощение разработки
Существует 2 функции:
1) История всех именений 
2) Доработка функционала изолированно 

Git (утилита) позволяет:
1) отслеживать изменения файлов
2) Сохранять их версии
3) Создавать ветки для параллельной работы
4) объединять изменения разных разработчиков и при необходмисти откатываться к прошлой версии
