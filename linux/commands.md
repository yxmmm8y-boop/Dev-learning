# Команды - теория

* cd - перейти в директорию
* mkdir - создание директории 
* rm/rm -rf - удалить файл/директорию (где "-r" значит РЕКУРСИВНО, а "-f" значит РЕКУРСИВНО)
* nano/vim - редактировать файл в терминале 
* touch - создать файл 
* chmod - изменить права
* sudo poweroff - выключить систему безопасно
* sudo systemctl suspend - спящий режим
* sudo reboot - перезагрузка
* sudo dnf upgrade - установка пакетов обновления системы 
* ls -l - просмотр прав доступа
* ls -ld - просмотр прав директории

## Подробно про команды, которые я не вдуплил  
- chmod - команда в Linux для *изменения прав доступа* к файлу или директории 

	Что делает? - задает:
	* кто может *читать* файл
	* кто может *изменять* файл
	* кто может *запускать* файл

	Виды записи прав на файл:
		
		# Символьный:
		
		Основные
		* r - read -- чтение
		* w - write -- запись
		* x - execute -- выполнение
		
		Пользователей:
		* u - user -- владелец файла
		* g - group -- группа
		* o - others -- остальные
		* a - all -- все (используется иногда)
		
			Пример:
			ls -l -- смотрим текущие права
			
			вывод:
			-rwxr-xr-- 1 ada ada 120 Mar 20 10:00 script.sh 
			нам нужно только "-rwxr-xr--", где:
			первый символ либо "-"(значит это файл) либо "d"(значит это директория
			следующие 9 символов - это права, по 3 на каждую группу:
			"rwx" - Права Владельца: может и читать, и писать, и выполнять
			"r-x" - Права Группы: может писать и выполнять
			"r--" - Права Остальных: могут только читать
		
		# Числовой:
		
		* r = 4
		* w = 2
		* x = 1

		Запись идёт тремя цифрами:
		
			chmod XYZ файл
			
			X - права пользователя
			Y - права группы
			Z - права остальных
		
		Примеры:

		* rwx = 4+2+1 = 7
		* rw- = 4+2 = 6
		* r-x = 4+1 = 5
		* r-- = 4
		* --- = 0
	
	Виды записи прав на директорию:
		
		r (4) - Позволяет читать содержимое папки, то есть видеть список файлов
		w (2) - позволяет создавать, удалять и переименовывать папки
		x (1) - позволяет заходить в папку и обращаться к обектам внутри
