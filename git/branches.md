# Git Branches

## Теория

 *Ветка* — это отдельная линия разработки, позволяющая работать независимо от основной версии проекта
 
 *Основная ветка* - main

###Задачи:
 - разработка новых функций
 - тестирование
 - изоляция изменений

## Работа с Ветками

 git branch - показывает ветки

 git branch feature - создает ветку

 git switch feature - переходит в ветку
 
 git switch -c feature - создает и переходит в ветку

 git merge feature - объединяет ветки
